 Обзор проекта
БетонТест - это интерактивное веб-приложение для виртуальных испытаний прочности бетона. Проект позволяет студентам, инженерам и строителям наглядно изучать свойства различных классов бетона через имитацию лабораторных испытаний.

Цели проекта:
 Образовательная - демонстрация свойств бетона в интерактивной форме

Научная - симуляция реальных физических процессов разрушения

Практическая - инструмент для предварительного анализа конструкций

Доступность - бесплатная онлайн-платформа без установки ПО

Целевая аудитория
Студенты строительных специальностей

Инженеры-строители и проектировщики

Преподаватели вузов и колледжей

Специалисты строительных компаний

Любители, интересующиеся строительством

 Особенности:
 Основной функционал
Интерактивная таблица классов - 13 классов бетона от B10 до B90

Виртуальные испытания - анимированное тестирование на прочность

3D визуализация - куб бетона с тремя состояниями разрушения

Сравнительный анализ - 4 наглядных сравнения прочности

Адаптивный дизайн - полная поддержка мобильных устройств

Оффлайн-работа - не требует интернет-соединения после загрузки

 Визуальные элементы:
Анимированный прогресс-бар с градиентной индикацией

Интерактивный 3D куб с CSS-трансформациями

Цветовая кодировка классов по прочности

Плавные переходы и микроанимации

Интуитивный UI с понятной навигацией

 Технические характеристики
Производительность: 60 FPS анимации

Размер проекта: ~500 KB (без сжатия)

Время загрузки: < 2 секунд на 3G

Поддержка браузеров: Chrome 60+, Firefox 55+, Safari 11+, Edge 79+


Языки: Русский интерфейс, с возможностью локализации
